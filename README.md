# SPA web application built with CRA

This project was built with [Create React App](https://github.com/facebook/create-react-app).

*本地https自签名证书生成（生成self-signed的TLS certificate）
```
$ cd server
$ ls -al
$ chmod +x ./generate_cert.sh
$ ./generate_cert.sh
```

*本地开发和测试
```
yarn watch:client
yarn watch:server
```
